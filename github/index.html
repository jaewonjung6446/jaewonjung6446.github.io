Cleanup rules

공개 노출 기준

공개 유지

채용자가 열었을 때 역할, 문제, 구현, 결과가 1분 안에 보이는 저장소만 공개 유지합니다.

비공개 전환

빈 저장소, 과제성 저장소, 설명 없는 실험 저장소, 현재 포지션과 무관한 저장소는 비공개 또는 archive 처리합니다.

README 보강

각 공개 저장소 README 상단에 담당 역할, 기술 스택, 핵심 구현, 실행/검증 방법, 공개 범위 정책을 둡니다.
